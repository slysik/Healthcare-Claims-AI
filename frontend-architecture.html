SSE Streaming Data Flow

1. User Input

User types query in ChatPanel textarea → sendMessage(query) via useChat hook

↓

2. SSE Connection

POST /api/chat/stream with AbortController — ReadableStream + TextDecoder parses chunked JSON lines prefixed with data:

↓
Trace Events
{node, status, timing_ms}
→ setCurrentTrace()
→ live AgentTrace badges
Answer Chunks
{text}
→ setCurrentAnswer()
→ streaming preview text
Complete Response
AgentResponse
→ setMessages()
→ full MessageBubble
Error Events
{message, recoverable}
→ setError()
→ retry button UI
↓

4. MessageBubble Renders

Conditionally renders sub-components based on response_data properties: intent badge → AgentTrace → Markdown answer → SqlViewer → ResultsTable → ChartView → Citations

Component Details

◈ ChatPanel

  • Consumes entire useChat hook
  • Auto-resizing textarea (Enter send, Shift+Enter newline)
  • Suggested query buttons (pre-seeded)
  • Renders live streaming trace + preview during SSE
  • Cancel button while streaming, retry on error
  • Auto-scrolls to bottom on new messages

◈ MessageBubble

  • Intent badge: nl2sql (blue), rag (green), clarify (amber)
  • Timing display + SQL retry count
  • ReactMarkdown with remark-gfm for answer text
  • Conditionally renders sub-components by data presence

◈ AgentTrace

  • Renders node execution as colored badges
  • Status icons: spinner (running), check (complete), x (error)
  • Timing display in ms per node
  • Used both live (streaming) and historical (in messages)

◈ SqlViewer

  • Collapsible SQL code block
  • Copy-to-clipboard button
  • Props: sql: string

◈ ResultsTable

  • Dynamic columns from object keys
  • Sortable column headers (asc/desc)
  • Max 50 rows displayed with count indicator
  • Props: data: Record<string, unknown>[]

◈ ChartView

  • Recharts: BarChart, LineChart, or PieChart
  • XAxis = 1st column, YAxis = 2nd column
  • Max 20 data points, 300px responsive container
  • Props: data + chartType: 'bar'|'line'|'pie'

◈ Citations

  • Expandable source list with count
  • Shows doc_name, page number, relevance score
  • Expand individual citation for full text
  • Props: citations: Citation[]

◈ UploadPanel

  • Drag-and-drop zone + file picker
  • Routes .csv → uploadCSV(), .pdf → uploadPDF()
  • Success/error toast notifications
  • Displays loaded datasets + indexed documents

Design Patterns

Hooks-Only State

No Redux/Zustand — useState + useRef + useCallback. All state local to components or lifted via useChat.

Conditional Rendering

MessageBubble renders sub-components only when matching data exists in AgentResponse (sql, results, citations).

ReadableStream SSE

Fetch API + ReadableStream + TextDecoder for chunked JSON parsing. AbortController for cancellation.

Demo Mode

DEMO_MESSAGES pre-loaded into useChat on init. Shows NL2SQL + RAG examples without requiring a backend.
